# Natural Earth 10m Land Shapefiles

This directory should contain the Natural Earth 10m physical land shapefiles used for overland detection.

## Download Instructions

1. Download the ZIP file from:
   https://www.naturalearthdata.com/http//www.naturalearthdata.com/download/10m/physical/ne_10m_land.zip

2. Extract the contents into this directory (`data/shp/ne_10m_land/`).

## Required Files

- `ne_10m_land.shp`
- `ne_10m_land.shx`
- `ne_10m_land.dbf`
- `ne_10m_land.prj`
